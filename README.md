# Chatter
Android Месенджер з можливістю відправки та отримання повідомлень у реальному часі та реєстрації по номеру телефону.
- Kotlin
- Clean Architecture
- MVI
- Jetpack Compose
- Firebase
- Room
- Hilt
- Preferences Data Store
- Coil
- Timber
- JUnit
- Espresso
